# homebrew-autofix

Homebrew tap for [autofix](https://github.com/trustabl/autofix) — AI-powered behavioral attestation and fix generation for Python agent tools.

## Install

```bash
brew tap trustabl/autofix
brew install autofix
```

## Usage

```bash
autofix fix --source ./my-tool --model gpt-4o
autofix version
```

For full documentation, see [github.com/trustabl/autofix](https://github.com/trustabl/autofix).
